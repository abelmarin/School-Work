Research

An HTML comment is a bit of code which does not do anything in the actual program but which is very important for documentation reasons. Comments do not show up in the browser and therefore only show up in the sourcecode. All HTML comments begin with <!-- and is bookended by -->.

The main purpose of comments in HTML is for documentation. This is when you are working on an HTML document and want to return to it later on. In this case you leave a comment in order to tell you, or another person working on the same docuemnt, what you were doing with the code. Another reason to use comments is to segment off you docuemnt. For instance, if you have a header, a navigation bar, a main content section, and a footer it would be worthwhile to include comments which tell you where each section starts.
